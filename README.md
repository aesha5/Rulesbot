# Rule-Based AI Chatbot 🤖

A simple rule-based chatbot built using Python as part of DecodeLabs Artificial Intelligence Project 1.

## Features

- Responds to predefined user inputs
- Uses dictionary-based response mapping
- Handles greetings
- Handles exit commands
- Runs continuously until user exits
- Provides default responses for unknown inputs

## Technologies Used

- Python 3

## How to Run

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to project folder

```bash
cd Rule-Based-AI-Chatbot
```

3. Run the chatbot

```bash
python chatbot.py
```

## Sample Interaction

```
You: hello
Bot: Hello, nice to meet you!

You: how are you
Bot: I am Great! What about you?

You: bye
Bot: Goodbye! Have a nice day.
```
