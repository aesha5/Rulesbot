# Project Report

## Project Title

Rule-Based AI Chatbot

## Objective

To create a chatbot that responds to predefined user inputs using rule-based logic.

## Implementation

The chatbot uses:

- Python Dictionary
- get() method
- while loop
- if condition for exit command

## Features

- Greeting responses
- Help command
- Exit command
- Unknown input handling

## Conclusion

This project demonstrates the fundamentals of Artificial Intelligence through predefined rules and decision-making logic.
